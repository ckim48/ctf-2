How to make a website

You need 3 different types of files to make a website. You need .html, .css, and .js. Html help users to see the basic structure of the site, while css is making the website look better. Js is for users to interact with the server or for deal with user data. After making basic things, you have to upload on to webserver, which makes user all around the world can access to your website.
